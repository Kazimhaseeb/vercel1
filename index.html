03 / Journey & Experience

Developer Timeline

A chronological journey through key milestones, technologies mastered, and projects shipped — from the first Python script to production trading systems handling live capital.

Key Milestones

1
Python & Automation — The Beginning
Started with Python fundamentals, building web scrapers, file automation scripts, and data processing utilities. Discovered the immense power of programmatic solutions.
2
Data Science & Financial Markets
Dove into pandas, NumPy, and Matplotlib for financial data analysis. Built the first OHLCV archiving pipeline and exploratory analysis notebooks using Polygon.io.
3
Machine Learning — Theory Meets Practice
Studied scikit-learn and TensorFlow, trained first LSTM models on stock data. Applied ML to real trading: price direction classification, anomaly detection, FinBERT sentiment scoring.
4
First Live Trading Bot (Alpaca)
Shipped the first production bot on Alpaca with paper trading. Implemented SMA crossover, RSI filter, trailing stops. A transformative milestone — watching automated systems trade with real logic.
5
Multi-Exchange Expansion
Extended to Binance, Bybit, IG Broker, and TastyTrade. Built grid engines, options bots, and WebSocket-driven real-time systems. Infrastructure grew to 10+ active production bots.
6
Backend Engineering — FastAPI & Django
Built REST APIs to serve trading data and expose bot controls. Mastered FastAPI async patterns, Django REST Framework, PostgreSQL, Redis, and Celery for production-grade web services.
7
RAG & LLM Systems
Adopted LangChain and ChromaDB to build document intelligence tools. Built AI Trading Research Assistant — a RAG system over SEC filings and earnings transcripts for investment research automation.
8
Flutter Mobile & Full-Stack Deployment
Expanded to Flutter/Dart for iOS and Android mobile apps, bringing the trading ecosystem to mobile. Established full CI/CD pipelines with Docker, GitHub Actions, and cloud deployment infrastructure.
