Automation Scripts Portfolio

Exchange Data Scraper: Async scraping of orderbook, funding rates, open interest, and liquidations across Binance, Bybit, OKX every 60 seconds.
Telegram Signal Bot: Real-time broadcast of trading signals, bot status updates, P&L summaries, and risk alerts to a private channel with command handling.
Daily Report Emailer: Styled HTML email with trade summary, drawdown metrics, best/worst trades — sent via SMTP at market close (4:05 PM ET).
News Headline Harvester: Scrapes financial news from Reuters, Bloomberg, CoinDesk — deduplicates, stores structured data for sentiment analysis input.
Celery Task Scheduler: Distributed task queue managing 20+ periodic jobs with retry logic, dead-letter queuing, and failure alerting via Telegram.
Excel Trade Journal: Auto-generates formatted trading journals and performance reports as styled Excel workbooks using openpyxl with charts.
SEC Filing Downloader: Bulk-fetches 10-K/10-Q filings from EDGAR API, parses XBRL data, and organises into structured datasets for ML ingestion.
Social Sentiment Tracker: Scrapy spider monitoring Reddit (r/wallstreetbets, r/stocks) for ticker mentions, sentiment scoring, and trend alerts.
Async Architecture

All scrapers use aiohttp + asyncio for concurrent HTTP requests, achieving 50x throughput vs synchronous code. aiohttp session pooling and rate-limit-aware semaphore controls prevent API bans.

Celery + Redis

Celery Beat manages 20+ periodic tasks — data scraping, report generation, model re-training, and cleanup jobs. Redis serves as both message broker and result backend with task monitoring via Flower.
