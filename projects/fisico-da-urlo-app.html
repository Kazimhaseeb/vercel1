Fisico Da Urlo is a custom QR-based attendance management system for gyms. Members scan a unique QR code to instantly record check-in and check-out — eliminating paper registers and reducing queues at the entrance.

The Flutter app connects to a Python FastAPI backend on Aruba CMP with a shared MySQL database. It works as the member-facing half of the full Fisico ecosystem, alongside the React.js admin panel used by gym staff for member CRUD and attendance oversight.

Part of the Fisico Da Urlo ecosystem — this mobile app pairs with the web admin dashboard (Project 02 · Web Apps) sharing the same FastAPI backend and database.

Core Features

QR Code Check-In — Members scan a unique QR code to instantly record attendance with no manual input.
Fast Check-In / Out — Secure, speedy entry and exit reducing queues and wait times at the gym.
Real-Time Logging — Attendance data recorded and visible immediately for gym management staff via the admin panel.
Mobile-First UI — Smooth, intuitive Flutter interface optimised for a seamless member experience.
Profile & History — Members view personal details, membership status, and full attendance history.
Operational Efficiency — Eliminates paper registers and manual errors, saving time for gym staff.
